# SEO Strategy — Hectar.ro

## Rezumat executiv

Site-ul Hectar țintește fermierii din România și Moldova care caută servicii de pulverizare cu drona. Strategia SEO combină cuvinte cheie tranzacționale (intenție de cumpărare) cu cuvinte cheie informaționale (conținut de jurnal) pentru a capta trafic la toate etapele procesului de decizie.

## 3. Structura On-Page SEO

### Meta Titles (format recomandat)
- Format: `[Beneficiu principal] · [Cuvânt cheie principal] · Hectar`
- Max: 60 caractere
- Include cuvântul cheie primar în primele 30 de caractere

### Meta Descriptions
- Format: Problemă → Soluție → CTA
- Include cuvântul cheie principal, 1–2 long-tail, și un număr concret (preț, suprafață, timp)
- Max: 155 caractere

### Structura H-urilor
- H1: 1 per pagină, include cuvântul cheie principal
- H2: Secțiuni principale, include variante ale cuvântului cheie
- H3: Subsecțiuni specifice

### Schema.org
- [x] Organization + LocalBusiness pe homepage
- [x] Service pe fiecare pagină de serviciu (cu Offer)
- [x] FAQPage pe /intrebari-frecvente
- [x] BreadcrumbList pe toate paginile interioare
- [x] Article pe fiecare postare de blog
- [ ] AggregateRating (de adăugat când avem recenzii)
- [ ] Review individual (de adăugat testimoniale)

## 4. Strategie Blog / Conținut Informațional

### Calendar editorial recomandat
- **Frecvență:** 2–4 articole/lună în sezon (mar–aug), 1/lună în extrasezon
- **Format:** articole de 800–1500 cuvinte, bazate pe experiențe reale
- **Structură:** Problemă → Context → Soluție → Acțiune

### Topice prioritare (în ordine)
1. ✅ Fuzarioza la grâu — timing de tratament
2. ✅ Mana viei — preventiv vs. curativ
3. ✅ Dronă vs. tractor — comparație onestă
4. ✅ Tratamente de primăvară la livadă
5. ✅ Raportul GPS — ce primești după tratament
6. ✅ Primul sezon cu drona la vie
7. Rapănul — calendar la meri (de scris)
8. Biostimulatorii — când chiar funcționează (de scris)
9. Sclerotinia la rapiță — cum o recunoaștem (de scris)
10. Tratamentele biologice cu drona (de scris)

## 5. SEO Tehnic

### Performanță (Core Web Vitals)
- [ ] LCP < 2.5s — imagini Unsplash cu `fetchpriority="high"` pe hero
- [ ] CLS < 0.1 — dimensiuni fixate pe imagini (width/height pe img)
- [ ] INP < 200ms — site static Astro, minimal JS
- [ ] Compresie HTML activată (compressHTML: true în astro.config)

### Crawling și indexare
- [x] robots.txt configurat corect
- [x] sitemap.xml generat dinamic cu toate URL-urile
- [x] llms.txt pentru AI engines
- [ ] Google Search Console — verificare și submitere sitemap
- [ ] Bing Webmaster Tools — verificare și submitere
- [ ] Indexare Google pentru noul domeniu

### Core tehnic
- [x] HTTPS (Netlify default)
- [x] Canonical tags pe toate paginile
- [x] noindex pe paginile private (/contact/trimis)
- [x] Favicon + Apple Touch Icon + Web Manifest
- [x] Open Graph tags (prin BaseLayout)
- [x] Schema.org JSON-LD pe toate paginile

## 6. Off-Page SEO — Construire autoritate

### Backlinks prioritare
1. **Directoare agricole românești** — APIA, MADR, site-uri de fitosanitare
2. **Furnizori de fitosanitare** — cerere de listing ca serviciu de aplicare
3. **Reviste agricole românești** — Ferma, Agrointeligenta, AgroTV
4. **Asociații de viticultori** — ONVV, Asociatia producatorilor de vin
5. **Site-uri cooperativele agricole** — schimb de link-uri cu parteneri locali
6. **Google Business Profile** — creare și optimizare profil

### Conținut guest
- Articole de expertiză în reviste agricole online
- Colaborări cu agronomi pe LinkedIn (piața Moldova + România)

## 7. Submitere la motoare de căutare

### La lansare:
1. Google Search Console → Proprietate nouă → Submitere sitemap
2. Bing Webmaster Tools → Submitere sitemap
3. Yandex (pentru Moldova) → Yandex Webmaster
4. Google Business Profile → Profil complet cu poze, descriere, categorii
5. AI engines: llms.txt deja creat → verificare indexare Claude, Perplexity, ChatGPT

### Timeline recomandat post-lansare:
- Ziua 1: Submitere Search Console + Bing
- Săptămâna 1: Google Business Profile complet
- Luna 1: Primele 2–3 backlinks calitative
- Luna 2–3: Conținut blog consistent, monitorizare GSC
- Luna 6: Evaluare poziționare, ajustare cuvinte cheie
